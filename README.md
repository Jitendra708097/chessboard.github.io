# Chessboard GitHub Pages

A simple static website demonstrating a chessboard UI built with HTML, CSS, and JavaScript. This repository is intended to be published via GitHub Pages.

## Demo
If GitHub Pages is enabled, the site will be available at: https://Jitendra708097.github.io/chessboard.github.io/

## Features
- Responsive chessboard layout
- Click / drag interactions (if implemented)
- Clean, minimal design for embedding or demos

## Local development
1. Clone the repository:

   ```bash
   git clone https://github.com/Jitendra708097/chessboard.github.io.git
   cd chessboard.github.io
   ```

2. Serve locally (example using a simple static server):

   ```bash
   npm install -g http-server
   http-server . -p 8080
   # then open http://localhost:8080 in your browser
   ```

Or open `index.html` directly in your browser for basic testing.
